## 🚀 Getting Started

1. **Clone the repository**

   ```bash
   git clone https://github.com/hul20/bilan-portfolio-1.git
   cd bilan-portfolio-1
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start development server**

   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
